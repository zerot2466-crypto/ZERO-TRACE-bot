## ⚙️ Installation & Configuration

### 1. Pré-requis
- Node.js LTS
- Git
- Un compte WhatsApp

### 2. Installation
```bash
git clone https://github.com/zerot2466-crypto/ZERO-TRACE-bot.git
cd ZERO-TRACE-bot
npm install
```

### 3. Configuration
Modifiez le fichier `config.js` avec vos propres clés et paramètres :
```javascript
const settings = {
  owner: ['22656354706'],
  prefix: '.',
  botName: 'ZERO TRACE',
  // Ajoutez vos clés API ici
};
```

### 4. Lancement
```bash
npm start
```
